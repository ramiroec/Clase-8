Recorrido y Métodos de Arreglos en JavaScript

Recorrer arreglos con bucle for

Podemos usar un bucle for tradicional para recorrer todos los elementos de un arreglo:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
let fLen = fruits.length; // Obtenemos la cantidad de elementos
let text = "<ul>";
for (let i = 0; i < fLen; i++) {
  text += "<li>" + fruits[i] + "</li>";
}
text += "</ul>";

Recorrer arreglos con forEach

El método forEach ejecuta una función por cada elemento del arreglo:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

let text = "<ul>";
fruits.forEach(myFunction);
text += "</ul>";

function myFunction(value) {
  text += "<li>" + value + "</li>";
}

Agregar elementos con push()

El método push() añade uno o más elementos al final del arreglo:

const fruits = ["Banana", "Orange", "Apple"];
fruits.push("Lemon"); // Agrega un nuevo elemento
fruits.push("Grapes", "Peach"); // Puedes agregar múltiples elementos
Nota: push() modifica el arreglo original y devuelve la nueva longitud.
